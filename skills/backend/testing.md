# Skill: Backend Testing

Use when adding backend validation.

## Guidance

- Add service-level tests for business behavior.
- Add controller or endpoint tests for API behavior.
- Cover happy path, validation, negative, and edge cases.
- Avoid tests that depend on external services unless integration scope is explicit.
- Keep test data small and readable.

## Outputs

- Unit tests.
- Integration tests when required.
- Build and test evidence.
