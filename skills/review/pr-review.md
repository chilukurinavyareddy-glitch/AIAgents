# Skill: PR Review

Use after PR creation or reuse.

## Guidance

- Verify source branch and target branch.
- Verify Jira link.
- Verify reviewers by stable Bitbucket identifiers where available.
- Verify validation evidence and execution score.
- Confirm no unsafe merge or deploy action happened.

## Outputs

- PR completeness result.
- Reviewer result.
- Merge readiness decision.
- PR comments to add.
