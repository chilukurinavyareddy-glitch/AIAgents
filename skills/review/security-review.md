# Skill: Security Review

Use during code review, PR review, and merge readiness checks.

## Guidance

- Check authentication, authorization, input validation, output encoding, secrets, logging, and dependency risks.
- Treat security blockers as merge blockers.
- Distinguish real reusable security issues from style preferences.
- Send reusable security mistakes to the learning agent.

## Outputs

- Security findings.
- Security blocker count.
- Learning candidates.
