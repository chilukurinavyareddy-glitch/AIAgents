# Skill: Merge Readiness

Use before merge automation.

## Guidance

- Validate execution score threshold.
- Confirm tests pass.
- Confirm code review and PR review pass.
- Confirm confidence is high.
- Confirm zero security blockers.
- Resolve only low-risk additive conflicts automatically.
- Block high-risk conflicts.
- Never deploy automatically.

## Outputs

- Merge decision.
- Conflict status.
- Post-merge validation plan.
