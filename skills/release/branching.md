# Skill: Branching

Use for feature branch and worktree operations.

## Guidance

- Create feature branches from the configured base branch, normally `development`.
- Use one branch and one worktree per real feature.
- Reuse existing branches when they match the Jira issue and feature slug.
- Do not rebase automatically.
- Do not force push.

## Outputs

- Branch name.
- Worktree path.
- Reuse/create result.
