# Skill: Frontend Forms

Use when implementing forms, validation, and form-driven workflows.

## Guidance

- Validate required fields.
- Show loading and error states.
- Prevent duplicate submit while a request is pending.
- Keep validation messages understandable.
- Avoid trusting client-side validation as a security boundary.
- Do not expose secrets or sensitive data in the UI.

## Outputs

- Form controls.
- Validation logic.
- Error display.
- Tests for valid and invalid states.
