# Skill: Jira Creation

Use when Jira work creation is allowed by policy.

## Guidance

- Run duplicate-prevention preflight before creation.
- Search existing open and completed work.
- Reuse open matching work.
- Ask before creating new work when a completed match exists.
- Do not auto-close unrelated Jira items.

## Outputs

- Jira preflight result.
- Created or reused issue list.
- Lifecycle sync notes.
