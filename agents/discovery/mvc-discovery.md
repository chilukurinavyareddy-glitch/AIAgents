# Discovery Plugin: ASP.NET MVC

Plugin id: `mvc-discovery`

Use when the project contains ASP.NET MVC structure.

Detect:

- controllers
- actions and routes
- areas
- Razor views and partials
- filters, model binders, validation attributes
- view models and service dependencies

Output notes:

- map controller actions to future APIs
- map views to future Angular screens
- identify auth and authorization attributes
