# Discovery Plugin: ExtJS

Plugin id: `extjs-discovery`

Use when the application uses ExtJS.

Detect:

- `Ext.application`
- controllers, views, view models, stores, models
- routes, menus, grids, forms, and custom components
- validation rules and permission checks embedded in UI code
- API endpoints called by stores or proxies

Output notes:

- create screen map candidates for rewrite planning
- identify grid, form, and workflow parity requirements
- flag tightly coupled UI/business logic
