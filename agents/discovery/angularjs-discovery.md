# Discovery Plugin: AngularJS

Plugin id: `angularjs-discovery`

Use when the application uses AngularJS 1.x.

Detect:

- module declarations
- controllers, directives, services, filters
- route/state configuration
- templates and partials
- legacy dependency injection naming

Output notes:

- map AngularJS controllers to future Angular feature components
- identify hidden validation and permission logic
- flag global scope and digest-cycle risks
