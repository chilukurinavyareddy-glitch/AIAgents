# Discovery Plugin: Angular

Plugin id: `angular-discovery`

Use when the application contains Angular workspace files.

Detect:

- `angular.json`
- `package.json`
- routes, standalone components, guards, interceptors
- feature folders, shared components, services, and environments
- test setup and production build output

Output notes:

- identify shared UI locks such as routing, app config, layout, navigation, auth guards, and interceptors
- identify API base URL configuration
- identify build and test commands
